1. Какие данные мы собираем

Для обработки заказа мы можем собирать:

  • Ваше имя и username (из Telegram, если вы зашли через мини-приложение)
  • Адрес электронной почты (если вы его указали)
  • Номер телефона (для оплаты через СБП)
  • Данные о платеже (сумма, дата, статус)

2. Как мы используем данные

  • Для обработки и доставки заказа
  • Для связи с вами по вопросам оплаты
  • Для предотвращения мошенничества
  • Для улучшения работы сервиса

3. Передача данных третьим лицам

Мы передаём данные только платёжным системам (Lava, Enot.io) для обработки платежей. Ваши данные не продаются и не передаются рекламным сетям.

4. Хранение данных

Данные о заказах хранятся 12 месяцев. Вы можете запросить удаление ваших данных, написав на ksioneru@gmail.com.

5. Ваши права

  • Запросить копию ваших данных
  • Исправить неточные данные
  • Потребовать удаления данных
  • Отозвать согласие на обработку
